# Chat Application

Chat Application is a full-stack real-time messaging platform that enables secure one-to-one communication between friends. It features JWT-based authentication, OTP-based password reset, real-time messaging with Socket.IO, friend management, profile customization, media sharing, and live online status tracking

## Features

* User Registration, Login, Logout & OTP-Based Password Reset
* Secure JWT Authentication & Password Encryption (bcrypt)
* User Profile Management with Profile Photo Upload
* Search Users, Send Friend Requests & Manage Friends
* One-to-One Real-Time Messaging using Socket.IO
* Messaging: Edit, Delete, Reply, Forward, Pin & Emoji Reactions
* Image & Voice Message Sharing
* Online/Offline Status, Last Seen, Typing Indicators & Unread Messages
* Responsive UI with Light/Dark Theme 

## Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB
* Socket.IO
* JWT (JSON Web Token)
* bcrypt
* Postman
