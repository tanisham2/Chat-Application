# Chat Application

Chat Application is a full-stack real-time messaging platform that enables users to communicate instantly through one-to-one chats. The application supports user authentication, real-time messaging using Socket.IO, user search, online/offline status tracking, and secure communication with JWT authentication.

## Features

* User registration, login, logout
* JWT Authentication
* Password Encryption using bcrypt
* User Profile Management
* View All Users
* Search Users
* One-to-One Chat
* Real-Time Messaging with Socket.IO
* Online/Offline Status
* Message Timestamps
* Delete Own Messages
* Image Upload Support

## Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB
* Socket.IO
* JWT (JSON Web Token)
* bcrypt
* Postman


## Running the Application

### Start Backend Server

```bash
cd server
npm run dev
```

### Start Frontend

```bash
cd client
npm start
```
